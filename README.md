
# InsuranceLandingPage

Este proyecto es una Landing Page moderna y responsiva diseñada para una compañía de seguros. Desarrollado utilizando **React** con **Vite** como herramienta de desarrollo y **Tailwind CSS** para el diseño, este sitio web ofrece una experiencia fluida y visualmente atractiva para los usuarios interesados en los servicios de seguros.

## Características

- **Diseño Moderno y Responsivo**: Optimizado para dispositivos móviles, tabletas y escritorios.
- **Componentización**: Componentes reutilizables para una estructura de código limpia y escalable.
- **Navegación Suave**: Scroll animado entre secciones.
- **Secciones Destacadas**:
  - **Encabezado**: Barra de navegación interactiva con efecto de cambio en el scroll.
  - **Servicios**: Presentación detallada de los servicios ofrecidos, con transiciones animadas.
  - **Testimonios**: Opiniones reales de clientes satisfechos.
  - **Footer**: Información de contacto y enlaces útiles.

## Tecnologías Utilizadas

- **React**: Librería para construir interfaces de usuario.
- **Vite**: Herramienta de desarrollo rápida y ligera para React.
- **Tailwind CSS**: Framework de diseño para estilos rápidos y consistentes.

1. Clona este repositorio:

   ```bash

   cd Insurance-LandingPage

2. Instala las dependencias:

   ```bash
   npm run install

3. Inicia el servidor de desarrollo:

   ```bash
